# Installation

## Windows
1. Install scoop
2. Install Fira Code Mono Nerd Font
3. setup wezterm
    - Install starship
    - Install nushell
    - Install yazi
        - Install "file"
4. setup nvim
5. install glazeWM and zebar (for windows style)
## Unix

1. Install Fira Code Mono Nerd Font
2. setup wezterm
    - Install starship
    - Install nushell
    - Install yazi
        - Install "file"
3. setup nvim
4. setup tmux
